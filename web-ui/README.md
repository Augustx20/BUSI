BusI – Plataforma de Gestión de Transporte 
Descripción del Proyecto

BusI es una plataforma web y móvil diseñada para optimizar la gestión del transporte escolar. El sistema conecta instituciones educativas, choferes y familias a través de una plataforma centralizada que permite organizar rutas, monitorear recorridos y mejorar la comunicación entre los distintos actores del servicio.

El objetivo principal del proyecto es digitalizar procesos que actualmente se gestionan de forma manual o informal, mejorando la seguridad, la trazabilidad y la eficiencia del transporte de estudiantes.

La plataforma permitirá visualizar el recorrido de los micros en tiempo real, registrar paradas, enviar notificaciones automáticas y gestionar la información de usuarios y rutas desde un panel administrativo.

Objetivo del Proyecto

El objetivo de BusI es desarrollar una plataforma que permita gestionar de forma integral el transporte escolar, incorporando herramientas tecnológicas que faciliten:

la organización de rutas escolares

el seguimiento en tiempo real de los vehículos

la comunicación entre padres, choferes e instituciones

el control de asistencia y paradas del recorrido

El sistema busca mejorar la seguridad de los alumnos y brindar mayor transparencia en el servicio de transporte.

Público Objetivo

El sistema está diseñado para tres tipos principales de usuarios:

Instituciones educativas

Administran el sistema, gestionan rutas, micros, choferes y alumnos.

Choferes o empresas de transporte

Gestionan recorridos, notifican eventos y registran el avance del viaje.

Padres o tutores

Pueden visualizar el recorrido del micro, recibir notificaciones y mantenerse informados sobre los traslados de sus hijos.

Estos actores interactúan dentro de la misma plataforma para mejorar la organización y el control del transporte escolar.

Problema que Resuelve

En muchas instituciones educativas el transporte escolar se gestiona con herramientas limitadas o de forma informal, lo que genera:

falta de trazabilidad de los recorridos

poca comunicación entre choferes y padres

escasa visibilidad del estado del viaje

dificultades para organizar rutas y horarios

BusI propone una solución digital que centraliza toda la información del servicio de transporte en un único sistema.

Funcionalidades Principales

El sistema incluye las siguientes funcionalidades:

Gestión de usuarios

Registro y autenticación de padres, choferes e instituciones educativas.

Gestión de rutas

Creación y administración de rutas escolares, asignando alumnos y choferes a cada recorrido.

Seguimiento en tiempo real

Visualización del micro en un mapa interactivo mediante geolocalización GPS.

Notificaciones y comunicación

Sistema de alertas automáticas y mensajería entre padres, choferes y escuelas.

Panel de administración

Interfaz de control para gestionar usuarios, recorridos y actividad del sistema.

Seguridad

Control de acceso por roles y protección de datos mediante autenticación segura.

Arquitectura del Sistema

El sistema se divide en dos partes principales:

Backend

Responsable de la lógica del sistema, gestión de datos, autenticación y comunicación con servicios externos.

Tecnologías principales:

Java

MySQL

APIs de geolocalización

Frontend

Responsable de la interfaz visual y la interacción con el usuario.

Tecnologías principales:

React

HTML

CSS

JavaScript

El frontend se comunica con el backend mediante una API que permite consultar rutas, usuarios, notificaciones y ubicación de los micros.

Concepto de la Landing Page

La landing page será la puerta de entrada al sistema BusI.

Su objetivo es explicar el funcionamiento del servicio y presentar sus beneficios a los usuarios.

El diseño conceptual propone representar visualmente un recorrido de transporte escolar visto desde arriba.

A medida que el usuario navega por la página, un micro avanza por una ruta mostrando diferentes paradas. Cada parada representa una funcionalidad del sistema, como el seguimiento en tiempo real, la gestión de rutas o la comunicación entre usuarios.

Este enfoque visual permite explicar el funcionamiento del sistema de forma dinámica y fácil de comprender.

Estructura de la Landing

La página estará organizada en las siguientes secciones:

Presentación del sistema

Problema actual del transporte escolar

Solución propuesta por BusI

Explicación del funcionamiento de la plataforma

Beneficios del sistema

Invitación a utilizar la plataforma

Cada sección ayudará a explicar progresivamente el valor del sistema.
